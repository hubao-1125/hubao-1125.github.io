安装Sui&连接本地网络

约 380 字大约 1 分钟

Sui

2025-04-21

最近在学习Sui，咋说呢。官方文档和官方中文文档差点儿意思。自己踩了几个坑之后，总结一下。

本人是Mac，所以用的是brew。

安装Sui

brew tap mystenlabs/tap

brew install mystenlabs/tap/sui

brew install cmake

brew install cmake-docs

brew install rust

启动本地网络

执行完上面的安装之后，执行下面的命令

启动本地网络

RUST_LOG="off,sui_node=info" sui start --with-faucet --force-regenesis

访问本地全节点

curl --location --request POST 'http://127.0.0.1:9000' \
--header 'Content-Type: application/json' \
--data-raw '{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "sui_getTotalTransactionBlocks",
  "params": []
}'

如果网络成功起来的话，会返回

{
    "jsonrpc": "2.0",
    "result": 168,
    "id": 1
}

将 Sui 客户端 CLI 连接到你的本地网络

替换Sui CLI 的RPC地址

sui client new-env --alias local --rpc http://127.0.0.1:9000

将活动环境设置为创建的新的环境

sui client switch --env local

检查当前活动环境

sui client active-env

显示当前活动地址

sui client active-address

这种情况一般会返回一个地址 0xd32199c50db8b03f2668312ba7003e2a19ed37ee3165b3013d779b2dd525037c

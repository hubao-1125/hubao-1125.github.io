Premium Index（溢价指数）

约 1827 字大约 6 分钟

合约

2025-10-27

Premium Index（溢价指数） 是加密货币永续合约中用于计算资金费率（Funding Rate）的一个关键指标，它反映了永续合约价格相对于现货市场加权平均价格（Price Index）的溢价或折价情况。

1. Premium Index 公式的定义

公式： Premium Index (P) = [ Max(0, Impact Bid Price − Price Index) − Max(0, Price Index − Impact Ask Price) ] / Price Index

  • Premium Index (P)：

    • 表示永续合约价格与现货市场价格（Price Index）之间的相对偏差。
    • 用于计算资金费率，帮助平衡多空市场，防止永续合约价格长期偏离现货价格。
    • 正值表示合约价格有溢价（多头占优），负值表示折价（空头占优）。

  • 公式中的组成部分：

    • Price Index：现货市场的加权平均价格（基于多个交易所的BTC/USDT价格）。
    • Impact Bid Price：基于订单簿的“冲击买入价”，即以一定交易量（如10万USDT）能买入的最大价格。（平均成交价）
    • Impact Ask Price：基于订单簿的“冲击卖出价”，即以一定交易量能卖出的最小价格。（平均成交价）
    • Max(0, x)：取值为 x 和 0 之间的最大值，确保偏差不会抵消为负。

  • 公式分解：

    • Max(0, Impact Bid Price − Price Index)：衡量永续合约的买入价格（Impact Bid Price）相对于现货价格的溢价。如果 Impact Bid Price 高于 Price Index，则返回正值；否则为0。
    • Max(0, Price Index − Impact Ask Price)：衡量永续合约的卖出价格（Impact Ask Price）相对于现货价格的折价。如果 Impact Ask Price 低于 Price Index，则返回正值；否则为0。
    • 差值除以 Price Index：将偏差标准化为相对比例，反映溢价/折价的程度。

2. Premium Index 的作用

  • 资金费率的基础：
    • Premium Index 是资金费率计算的核心部分，通常结合固定利率（如0.01%）得出最终资金费率：
      • Funding Rate = Premium Index + Clamp(Interest Rate − Premium Index, −0.05%, 0.05%)
      • （Clamp 表示将差值限制在−0.05%到0.05%之间。）
    • 正 Premium Index 导致正资金费率（多头支付空头），负 Premium Index 导致负资金费率（空头支付多头）。
  • 平衡多空市场：
    • 当合约价格高于现货（正溢价），多头需支付费用，激励空头入场，压低合约价格。
    • 当合约价格低于现货（负折价），空头支付费用，激励多头入场，推高合约价格。
  • 锚定现货价格：
    • 通过资金费率调整，Premium Index 确保永续合约价格长期接近现货价格（Price Index）。

3. 计算 Premium Index（含负数情况）

让我们通过示例逐步计算 Premium Index，并分析 Basis 为负数（Last Price < Price Index）的情况如何影响：

示例数据

  • Price Index = 50,000 USDT（现货市场加权平均价格）。
  • Impact Bid Price = 50,100 USDT（以10万USDT交易量能买入的最大价格）。
  • Impact Ask Price = 50,200 USDT（以10万USDT交易量能卖出的最小价格）。
  • Last Price = 49,800 USDT（永续合约最新成交价格，Basis = 49,800 − 50,000 = −200 USDT，负值）。

步骤 1：计算溢价部分

  • Max(0, Impact Bid Price − Price Index)：
    • Impact Bid Price − Price Index = 50,100 − 50,000 = 100 USDT。
    • Max(0, 100) = 100 USDT（正值，表示买入价有溢价）。

步骤 2：计算折价部分

  • Max(0, Price Index − Impact Ask Price)：
    • Price Index − Impact Ask Price = 50,000 − 50,200 = −200 USDT。
    • Max(0, −200) = 0 USDT（负值取0，无折价）。

步骤 3：计算 Premium Index

  • Premium Index = (100 − 0) / 50,000
    • = 100 / 50,000
    • = 0.002 = 0.2%（正值）。

负 Basis 场景

  • Basis = Last Price − Price Index = 49,800 − 50,000 = −200 USDT（负值）。
  • 负 Basis 表示永续合约价格低于现货价格，可能导致负资金费率，但 Premium Index 基于订单簿的 Impact Bid/Ask Price，而不是 Last Price。
  • 如果 Impact Bid Price 和 Impact Ask Price 也低于 Price Index（例如，Impact Bid = 49,900 USDT，Impact Ask = 49,950 USDT）：
    • Max(0, 49,900 − 50,000) = Max(0, −100) = 0 USDT（无溢价）。
    • Max(0, 50,000 − 49,950) = Max(0, 50) = 50 USDT（有折价）。
    • Premium Index = (0 − 50) / 50,000 = −50 / 50,000 = −0.001 = −0.1%（负值）。

结论：

  • Premium Index 可以为正、负或零，具体取决于 Impact Bid/Ask Price 相对于 Price Index 的偏差。
  • 负 Basis（Last Price < Price Index）通常对应订单簿价格偏低，可能导致负 Premium Index，从而引发负资金费率（空头支付多头）。

4. Premium Index 与标记价格的关系

标记价格公式： Mark Price = Median(Price1, Price2, Last Price)，其中：

  • Price1 = Price Index × (1 + Last Funding Rate × Time Until Next Funding / Funding Period)

  • Price2 = Price Index + Moving Average (30 min Basis)

  • Premium Index 的间接影响：

    • Premium Index 影响资金费率（Funding Rate），而资金费率影响 Price1。
    • 负 Premium Index 通常导致负资金费率，使 Price1 略低于 Price Index（例如，Price1 = 50,000 × (1 − 0.0001 × 0.5) = 49,997.5 USDT）。
    • 负 Basis（Last Price < Price Index）使 Price2 偏低（Price2 = 50,000 + (−200) = 49,800 USDT）。
    • 标记价格通过取中位数，平衡这些因素，通常接近但略低于 Price Index。

5. 负 Premium Index 的影响

  • 资金费率：
    • 负 Premium Index（如−0.1%）导致负资金费率，空头需支付费用给多头，激励多头入场，推高合约价格。
    • 示例：仓位0.1 BTC，标记价格50,000 USDT，资金费率−0.01%，空头支付0.1 × 50,000 × 0.01% = 0.5 USDT/周期。
  • 交易策略：
    • 负 Premium Index 对多头有利（收到资金费率补偿），对空头不利（需支付费用）。
    • 交易者可利用负 Premium Index 做多永续合约，同时在现货市场做空，赚取资金费率和价格回归差价。
  • 标记价格：
    • 负 Premium Index 通过负资金费率和负 Basis 影响 Price1 和 Price2，使标记价格偏低，反映市场折价。

6. 实际应用场景

  • 套利机会：
    • 负 Premium Index（合约折价）提示买入永续合约、卖出现货的套利机会，等待价格回归。
    • 正 Premium Index（合约溢价）提示卖出永续合约、买入现货。
  • 风险管理：
    • 负 Premium Index 可能导致负资金费率，多头可从中获利，但需警惕价格进一步下跌。
    • 空头需考虑资金费率成本，设置止损以控制风险。
  • 监控 Premium Index：
    • 交易所（如Binance、Bybit）通常显示实时 Premium Index 和资金费率，交易者可参考调整仓位。

7. 注意事项

  • Impact Bid/Ask Price 的定义：
    • 不同交易所对 Impact Bid/Ask Price 的计算方式不同，通常基于固定交易量（如10万USDT）的订单簿深度。
    • 需查阅平台文档（如Binance的“资金费率计算”页面）了解具体算法。
  • 市场波动：
    • 极端市场中，Impact Bid/Ask Price 可能与 Last Price 差异较大，影响 Premium Index 的准确性。
  • Basis 与 Premium Index：
    • 负 Basis（Last Price < Price Index）通常与负 Premium Index 相关，但 Premium Index 更依赖订单簿数据，而非单一 Last Price。

8. 总结

  • Premium Index 衡量永续合约价格相对于现货价格（Price Index）的溢价/折价，基于订单簿的 Impact Bid/Ask Price 计算。
  • 公式：P = [ Max(0, Impact Bid Price − Price Index) − Max(0, Price Index − Impact Ask Price) ] / Price Index。
  • 负 Basis（Last Price < Price Index）可能导致负 Premium Index，引发负资金费率（空头支付多头）。
  • 负 Premium Index 使标记价格偏低，影响未实现盈亏和强平点，对多头有利（资金费率补偿），对空头增加成本。

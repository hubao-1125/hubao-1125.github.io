# padding


padding是定义元素边框与元素内容之间的空间，即上下左右的内边距。

可以单独设置，也可以一次性改变所有。有1～4个值可调整。

## 单独设置

`padding-top`
```html
padding-top:100px;
调整上内边距100px
```


`padding-bottom`
```html
padding-bottom:100px;
调整下内边距100px
```

`padding-right`
```html
padding-right:80px;
调整右内边距80px
```

`padding-left`
```html
padding-left:40px;
调整左内边距40px
```


## 简写设置


`四简写`
```html
padding: 30px 60px 90px 120px;
上内边距为30px
右内边距为60px
下内边距为90px
左内边距为120px
简写四个边的`px`设置，以顺时针方向设置，按上、右、下、左。
```



`三简写`
```html
padding:50px 100px 150px;
上内边距为50px
左右内边距为100px
下内边距为150px

```

`二简写`
```html
padding:50px 100px;
上下内边距为50px
左右内边距为100px
```

`一简写`
```html
padding:50px;
4个内边距都是50px

```

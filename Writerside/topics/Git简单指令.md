# Git简单指令

### git status
查看当前仓库下`git`的变更记录

### git add .
当前所有变更的文件添加

### git commit -a
提交变更

### git push
推送

### git pull
拉取更新

### git remote -v
查看当前本地`git`仓库远程remote

### git remote set-url
修改仓库远程remote
```git
假设我新的remote地址是xxx.test，那么指令就是
git remote set-url origin xxx.test
```

### git branch
查看当前分支

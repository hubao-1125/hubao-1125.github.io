# margin

margin是定义外边距的，定义元素周边空间，上下左右。

可以单独设置，也可以一次性改变所有。有1～4个值可调整。

## 单独设置

`margin-top`
```html
margin-top:100px;
调整上边距100px
```


`margin-bottom`
```html
margin-bottom:100px;
调整下边距100px
```

`margin-right`
```html
margin-right:80px;
调整右边距80px
```

`margin-left`
```html
margin-left:40px;
调整左边距40px
```


## 简写设置


`四简写`
```html
margin: 30px 60px 90px 120px;
上边距为30px
右边距为60px
下边距为90px
左边距为120px
简写四个边的`px`设置，以顺时针方向设置，按上、右、下、左。
```



`三简写`
```html
margin:50px 100px 150px;
上边距为50px
左右边距为100px
下边距为150px

```

`二简写`
```html
margin:50px 100px;
上下边距为50px
左右边距为100px
```

`一简写`
```html
margin:50px;
4个边距都是50px

```
